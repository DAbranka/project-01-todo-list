# Todo List - Junior Project

A simple todo list application built as a junior developer exercise to practice fundamental web development skills.

## Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Styling and layout
- **JavaScript (Vanilla)** - Core functionality and DOM manipulation
- **Vite** - Fast build tool and development server
- **pnpm** - Package manager

## About

This project is part of a junior developer training exercise focused on building foundational skills in:
- DOM manipulation
- Event handling
- Basic CRUD operations
- Component-based architecture
- Modern development tooling

## Getting Started

Install dependencies:
```bash
pnpm install
```

Run development server:
```bash
pnpm run dev
```

Build for production:
```bash
pnpm run build
```
